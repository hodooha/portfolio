강아지 용품 쇼핑몰 - 호두몰2

React와 Spring Boot를 사용하여 RESTful API를 구현한 반려견 용품 쇼핑몰 프로젝트입니다.
※React와 NodeJS 기반의 호두몰1을 React와 Spring Boot로 마이그레이션
기간/인원 : 24.09 ~ 2024.11 / 1인

개발환경

  • Language: Java 17
  • Frame work : Spring Boot(Backend), React(Frontend)
  • DB : MongoDB, Redis, RabbitMQ(Messaging)
  • Security & Auth : Spring Security, JWT Authentication, Google OAuth
  • Tools : IntelliJ IDEA, Visual Studio Code, Gradle, Docker

주요기능

  • 회원가입, 로그인, 구글로그인
  • 상품 조회/검색/구매, 카트/주문내역/쿠폰 조회
  • Admin : 상품 및 쿠폰 등록/수정/삭제, 주문내역 조회 및 수정

적용 기술 및 주요 구현 기능

Spring Security, JWT, OAuth2, Redis - 인증 및 권한 관리
  • Spring Security, JWT를 사용하여 인증 및 인가를 구현했고, OAuth2로 구글 로그인을 추가했습니다.
  • Redis 기반 Refresh Token 관리와 HttpOnly 쿠키를 활용하여 Replay Attack을 방지하고 인증 시스템의 보안을 강화했습니다.
  • TTL을 설정하여 Redis에 저장된 만료된 토큰을 자동으로 삭제하고, Access Token 갱신 속도를 최적화하여 성능을 개선했습니다.
  • Refresh Token Rotation(RTR)기법을 적용하여 탈취된 Refresh Token의 재사용을 차단하여 보안성을 강화했습니다.
Redis, RabbitMQ - 선착순 쿠폰 발급
  • 쿠폰 수량 관리를 Redis 캐시로 구현하고 Spring Scheduler로 DB와 동기화하여 응답시간을 단축하고 성능을 개선했습니다.
  • 쿠폰 발행을 RabbitMQ 메시지큐를 사용하여 비동기 처리함으로써 동시성 충돌을 피하고 안정성을 높였습니다.

쿠폰 발급 부하테스트 (Thread: 100, Ramp-up: 1, Loop: 10)
구분 응답시간(m/s) 처리량
(req/sec)
평균 최소 최대 표준편차
기존 501 10 4,973 784.56 125.3
Redis 125 8 355 73.15 430.7
RabbitMQ 210 10 560 121.13 315.3
Redis + RabbitMQ 181 8 712 126.06 340.8
  • Redis 캐싱 적용으로 평균 응답 시간을 501ms에서 125ms로 감소
  • RabbitMQ를 통한 비동기 처리로 처리량을 125.3req/sec에서 340.8req/sec로 개선
  • 캐싱만 적용했을 때 성능 수치는 가장 좋았지만, 메시지 큐를 함께 사용하여 동시성 문제를 해결하고 안정성을 높이기 위해 최종적으로 캐싱과 메시지 큐를 함께 적용
Docker - MongoDB 트랜잭션 처리를 위한 레플리카셋 구성
  • MongoDB 트랜잭션 기능 활용을 위해 Docker를 사용하여 레플리카셋을 구성했습니다.
  • Spring Boot MongoDB 트랜잭션으로 상품 주문 기능에서 원자성, 일관성, 독립성, 영속성을 보장했습니다.
Redux Toolkit - 상태 및 데이터흐름 관리
  • Redux Toolkit을 통해 상태 관리와 React 컴포넌트 간 데이터흐름을 효율적으로 구현했습니다.
  • Redux Thunk를 사용하여 비동기 데이터 처리를 간소화하고 상태 변경에 따른 UI 업데이트를 최적화했습니다.

React와 Express.js를 사용하여 RESTful API를 구현한 반려견 용품 쇼핑몰 프로젝트입니다.
기간/인원 : 23.12 ~ 2024.01 / 1인

개발환경

  • Language: JavaScript
  • Frame work : React(Frontend), Express.js(Backend)
  • DB : MongoDB(with Mongoose)
  • Auth : JWT Authentication, Google OAuth
  • Tools : Visual Studio Code

주요기능

  • 회원가입, 로그인, 구글로그인
  • 상품 조회/검색/구매, 카트/주문내역 조회, 이벤트 팝업창
  • Admin : 상품 등록/수정/삭제, 주문내역 조회 및 수정

적용 기술 및 주요 구현 기능

JWT, OAuth2 - 인증 및 권한 관리
  • JWT를 사용하여 인증 및 인가를 구현했고, OAuth2로 구글 로그인을 추가했습니다.
Redux - 상태 및 데이터흐름 관리
  • Redux를 사용하여 전역 상태를 효율적으로 관리하고, 상태 변경에 따른 UI 업데이트를 동적으로 구현했습니다.
  • Redux Thunk 미들웨어를 통해 비동기 데이터 처리를 간소화했습니다.
React Router - 상품 상세 조회
  • React Router를 통해 상품 클릭 시 상세 페이지로 이동하는 동적 라우팅을 구현하여 사용자 경험을 강화했습니다.
쿠키 관리 - 이벤트 팝업창 제어
  • 쿠키를 이용해 이벤트 팝업창의 "오늘 하루 보지 않기" 기능을 구현했습니다.
  • 유저의 행동 데이터를 쿠키로 저장하고 페이지 로드 시 해당 쿠키를 읽어 동적으로 UI를 제어했습니다.
반응형 웹 디자인 - 화면 크기별 맞춤 렌더링
  • @media를 활용하여 화면 크기에 따라 메뉴 위치, 글자 크기, 이미지 크기 등을 조정했습니다.
  • 모바일, 데스크탑 등 다양한 디바이스 환경에서 최적의 사용자 경험을 제공했습니다.
