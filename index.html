주요기능

  • 우동소식 - 동네 소식 및 사건 공유
  • 우동모임 - 모임 결성 및 채팅
  • 고객센터 - 문의 질의 및 답변

개인프로젝트

강아지 용품 쇼핑몰 - 호두몰2

React와 NodeJS 기반의 프로젝트 호두몰1을 React와 Spring Boot 기반으로 마이그레이션한 프로젝트
기간/인원 : 24.09 ~ 2024.10 / 1인

개발환경

  • Language: Java 17
  • Frame work : Spring Boot(Backend), React(Frontend)
  • DB : MongoDB, Redis, RabbitMQ(Messaging)
  • Security & Auth : Spring Security, JWT Authentication, Google OAuth
  • Tools : IntelliJ IDEA, Visual Studio Code, Gradle

주요기능

  • 회원가입, 로그인, 구글로그인
  • 상품 조회 및 구매, 카트/주문내역/쿠폰 조회
  • Admin : 상품 및 쿠폰 등록/수정/삭제, 주문내역 조회 및 수정

적용 기술 및 주요 구현 기능

  • Spring Security, JWT를 사용하여 인증 및 인가를 구현했고, OAuth2를 사용하여 구글 로그인을 구현했습니다.
  • REST 기반으로 CRUD를 구현했고 Spring Data MongoDB를 사용하여 데이터를 다뤘습니다.
  • 선착순 쿠폰 발행 시 쿠폰 수량 감소 기능을 Redis 캐시로 구현하여 응답시간을 줄임으로써 성능 개선을 하였습니다.
  • 선착순 쿠폰 발행 시 쿠폰 발행 기능을 RabbitMQ 메시지큐를 사용하여 비동기 처리함으로써 동시성 충돌을 피하고 안정성을 높였습니다.
